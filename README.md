# SHFiles

###  설명

- SHFiles은 탐색기 (맥, 리눅스 개발예정) 대체용 파일 매니저
- 기존 SHCommanderX의 리뉴얼 버전으로 불필요하고 중복된 코드들 제거, 알고리즘의 변경으로 안정성 및 속도 향상
- SHFiles의 SH는 아들이니셜
- License는 GPL v3를 따름

### 개발환경

- GUI : wxWidgets 3.1.3 Unicode Build
- IDE : CodeLite 14.0.0
- Compiler : gcc 10.1.0
